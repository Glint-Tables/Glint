# Installation

## From Github

Add the following to your _composer.json_ before running `composer install`.

```bash
...
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/ajkrumholz.glint.git"
    }
],
...

"require": {
    ...
    "glint/glint": "dev-main"
}
```

## From Packagist

Installation from Packagist is not yet supported.
